
# Nopixel-Inspired-Lambo700
NoPixel inspired Lamborghini Aventador LP700.

# Install
* Drag and drop
* Ensure lp700

# For use of sounds
* Drag [aventadorv12] into your `resources` folder
* Ensure [aventadorv12] in your server CFG
